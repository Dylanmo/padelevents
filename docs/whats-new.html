What's New — Padel Events

Recent enhancements to help you find and add padel events faster.

At a glance

We've focused on mobile usability, reliable calendar exports, and better filters so you can find matches that fit your level and schedule.

New & Improved Features

  • Quick Event Preview: Upcoming events load on page open so you see matches immediately. (assets/js/app.js)
  • Club Filter Chips: Dynamic club pills loaded from the API for fast selection. (assets/js/api.js + assets/js/app.js)
  • Multi-dimensional Filters: Level, type, time-of-day and weekday filters to find the right games. (assets/js/filters.js)
  • Mobile-first Card UI: Touch-optimized event cards and responsive layout for phones. (assets/style.v1.css, assets/js/app.js)
  • Happening Soon: Today & Tomorrow events surfaced to join immediate matches. (assets/js/app.js)
  • Google Calendar Links: One-click adds with correct UTC timestamps. (assets/js/calendar.js)
  • ICS Export: Download .ics files for Apple/Outlook imports. (assets/js/calendar.js)
  • Timezone-safe dates: Bangkok → UTC conversion to ensure calendar accuracy. (assets/js/calendar.js, assets/js/utils.js)
  • Filter persistence helpers: Ready to save your selections across visits. (assets/js/utils.js)
  • Device-aware links: Platform-specific signup flow for better join experience. (assets/js/app.js)

Major problems we solved

  • Better mobile scanning: Card layout and larger touch targets reduce friction on phones. (assets/style.v1.css)
  • Correct calendar times: Timezone handling prevents mismatched event times in user calendars. (assets/js/calendar.js)
  • Cross-platform support: ICS files and Google links make adding events universal. (assets/js/calendar.js)
  • Faster, safer deploys: Firebase cache rules keep HTML fresh while serving versioned assets quickly. (firebase.json)

Where to verify

See implementation details in the listed files. For player-facing previews, open Players page or the main app at Browse Events.
