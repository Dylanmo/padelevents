## How to View

### Option 1: Firebase Serve (Recommended)

```bash
firebase serve
```

Then visit:

- `http://localhost:5000/previews/`
- `http://localhost:5000/previews/tennis-ball-versions.html`
- `http://localhost:5000/previews/tennis-ball-shape-variations.html`
- `http://localhost:5000/previews/tennis-ball-realistic.html`
- `http://localhost:5000/previews/font-variations.html`
- `http://localhost:5000/previews/tagline-font-variations.html`

### Option 2: Live Server Extension

Right-click on any `.html` file ‚Üí "Open with Live Server"

### Option 3: Direct File View

Open any `.html` file directly in your browser (file:// protocol)
